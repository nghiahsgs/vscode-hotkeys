# vscode-hotkeys
vscode hotkeys

+ Tìm kiếm trong file
```
ctrl + f
```

+ Tìm kiếm và thay thế trong file
```
ctrl + h
```

+ Tìm kiếm cả project
```
ctrl + shift + f
```

+ Tìm kiếm và thay thế cả project
```
ctrl + shift + h
```

+ Format code js theo eslint
```
shift + alt + f
```

+ tìm kiếm tên file
```
ctrl + p
gõ tên file vào
```

+ di chuyển nhanh tới 1 file
```
ctrl + p
:dòng số bao nhiêu vào
```

+ mở settings trên vs code
```
ctrl + ,
```
